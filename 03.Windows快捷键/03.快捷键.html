                      # Windows全局快捷键

                      # 01.『常用快捷键』

                      『Windows10常用』

                      • 『win+a』打开右下角操作中心，可以开启『蓝牙、无线、移动热点等』，还可以查看一些通知

                      • 『win+I』 打开Windows10下的**『设置』**,和我们用control打开的『控制面板』还是有点区别，但是功能好像

                        • 注意： 快捷键在不同电脑上不是万能的，比如在ASUS上可以用Alt+F4关掉页面，但是在其他版本电脑上可能就只能用Fn+Alt+F4了（所以，命令行才是王道😄）

                      • 『Win+X然后加F』WIN+X先一会，然后按F就可以了 ，Windows 快速打开要去卸载软件

                      • 『ctrl+win+D 』 新建另一个桌面，并且转到那里

                      • 『Ctrl+win+左或右』，可以在；领近的几个桌面中切换，『类似，Linux下工作台的概念，是的』

                      • 『Ctrl+win+F4 』 关闭当前桌面

                        • 注意：使用win+ctrl+f4快捷键可以关闭当前虚拟桌面，如果当前的虚拟桌面有窗口，则窗口全部跑到前一个虚拟桌面上去

                      • 『Win+Tab』在有2个及以上桌面的时候『工作台』的时候，就能宏观看，但是如果只有1个桌面，就是用来看程序的个数，就退化为『Alt+Tab』快捷键

                      • 『win+shift+S』Windows10 内置截图功能，截图后，他就在剪切板中了，直接粘贴到需要的地方就行了

                        • 记忆方式：联想QQ的截图，顺时针旋转15°左右就是这3个键『这个功能好像是office的note带的功能！！』

                      • shift+F10 鼠标右击的快捷键

                      • ALT+TAB切换

                      • ALT+ESC切换

                      • ALT+空格键窗口菜单

                      • CTRL+ESC开始菜单

                      • 拖动某一项时按CTRL复制所选项目

                      • 拖动某一项时按CTRL+SHIFT创建快捷方式

                      • 『win+E』 打开资源管理器

                      • 『win+break』打开系统属性

                      • shift常按不放

                        • 1、在放入CD的时候，可以跳过自动播放CD
                        • 2、打开word时，可以跳过自启动的宏

                      • Alt+F4 关闭当前应用程序

                      • Alt+enter 查看文件夹属性、dos窗口的全屏和非全屏

                      • Ctrl+F4 关闭当前应用程序的当前文件（比如word）

                      # PrtSc键（Print Screen）

                      • 1、『PrtSc』只要按下这个PrtSc键那么就可以在文本中粘贴 你按下此键 就会在win默认的剪贴板里暂存了刚才的屏幕截图.这时候你打开win自带的画图工具,然后按『ctrl+v』,就会把你刚才截到的图粘在画图里了.这个功能主要是为了方便那些没有办法截到的动态片刻例如电影的某个场景.（不能在“照片”中找到截下的图）

                      • 2、『win+ PrtSc』不仅可以上面那么做，而且能够在“照片”中找到截下的图 注意：以上是截全屏

                      • 3、『Alt+ PrtSc』则是对当前窗口进行截图。如你打开“我的电脑”后，用此法就截取“我的电脑”窗口的内容。（不能在“照片”中找到截下的图）

                      Print Screen /Sysrq 打印屏幕/系统请求

                      # Scroll Lock（屏幕）滚动锁定

                      scroll lock就是“滚动锁”的意思，此键在win系统中一般不用，但是部分软件偶尔会使用到，比方说EXCEL办公软件，如果Scroll Lock关闭的情况，使用翻页键Page Down与Page Up时，单元格选定区域会随之发生移动。如果大家按下了Scroll Lock键后，则不会移动选定的单元格。

                      在电脑初期DOS系统时代，我们经常需要使用Scroll Lock键，由于屏幕显示是字符方式，每屏可以显示25行，每行有80个字符，该键的作用是查阅比较长的文本时进行翻滚，所以在DOS系统下很是有用，随着win系统可视操作系统的出现，大家查阅网页、文本都不需要键盘上scroll lock键了。

                      # Pause /Break 暂停/中断

                      System Request（系统请求）

                      • 滚动锁定Scroll Lock：灯亮时，按移动键时屏幕滚动；灯灭时，光标滚动；
                      • 暂停中断Pause Break：在想中断一些操作时，比如中断打印，则按此键，打印暂停。

                      • Num Lock（数字锁定）
                      • Scroll Lock（滚动锁定）
